# 📊 Desafio-Telecom-X2_IDG
Desarrollo desafío CIENCIA DE DATOS PYTHON ALURA LATAM - ORACLE ACADEMY

El **Desafío Telecom X parte 2** ofrece una oportunidad única para aplicar conocimientos fundamentales de 
**Estadisticas y Machine Learning, estadistica python probabilidad muestreo, relaciones-regresion-lineal, Regresión lineal Tecnicas avanzadas de Modelado, Clasificación de datos con Machine Learning, validación de modelos y métricas de evaluación y IA aumentada**, además de habilidades esenciales en ciencia de datos, en un escenario empresarial real.

---

## 🚀 Objetivo
Desarrollar modelos predictivos que permitan **anticipar el riesgo de cancelación de servicios (churn)** en una empresa de telecomunicaciones, apoyando la toma de decisiones estratégicas para reducir la pérdida de clientes.

---

## 🛠️ Metodología

1. **Preparación de datos**
   - Limpieza y transformación de variables.
   - Separación en conjuntos de entrenamiento y prueba.
   - Balanceo de datos para mejorar la representatividad.

2. **Análisis exploratorio**
   - Estadística descriptiva.
   - Visualización de distribuciones y tendencias.
   - Análisis de correlación entre variables.

3. **Modelado**
   - Aplicación de **regresión lineal** para comprender el impacto de cada variable en el churn.
   - Desarrollo de modelos de **machine learning** orientados a la predicción.
   - Evaluación mediante métricas de rendimiento (R², RMSE, precisión, etc.).

4. **Validación**
   - Uso de **validación cruzada** para asegurar la robustez de los modelos.
   - Comparación de resultados entre diferentes configuraciones y algoritmos.

---

## 📈 Resultados esperados
- Identificación de los factores más influyentes en la cancelación de servicios.
- Construcción de un modelo predictivo confiable para estimar el churn.
- Generación de insights accionables para la empresa, orientados a la **retención de clientes**.

---

## 🎯 Aprendizajes clave
- Aplicación práctica de conceptos estadísticos en un contexto empresarial.
- Importancia de la preparación y división adecuada de los datos.
- Uso de regresión lineal como herramienta interpretativa.
- Desarrollo de modelos de machine learning para problemas reales de negocio.
